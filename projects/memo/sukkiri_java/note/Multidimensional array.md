〜多次元配列について〜

1次元配列に縦の並びを加えると2次元配列になる。<br>
2次元配列は要素が縦横に並んだ表のようなもので、データを表のような形で扱いたい時に使用すると便利。<br>
2次元以上の配列を多次元配列という。<br>

2次元配列の宣言<br>
要素の型[][] 配列変数名 = new 要素の型[行数][列数];<br>

2次元配列の要素の利用<br>
配列変数名[行の添字][列の添字]<br>

例:兄弟二人の3科目のテスト結果を格納する2次元配列<br>

int[][] scores = new int[2][3]; // 2行3列の配列<br>
scores[0][0] = 40;<br>
scores[0][1] = 50;<br>
scores[0][2] = 60;<br>
scores[1][0] = 70;<br>
scores[1][1] = 80;<br>
scores[1][2] = 90;<br>
System.out.println(scores[1][1]);<br>

例:親配列とこ配列の要素数を表示<br>

int[][] scores = {{40, 50, 60}, {80,60,70}};<br>
// 2次元配列の初期化: 2次元配列の型[][] 配列変数名 = {{行},{列}};<br>
System.out.println(scores.length); //2が出力される(行の長さをみている為)<br>
System.out.println(scores[0].length); //3が出力される(列の長さをみている為)<br>

配列変数名[0].lengthで列の長さがみられる理由:先頭要素の数を見ているため<br>
