〜用語解説:ソースコードの基本構造〜

{}で囲まれた部分…ブロック
外側のブロック…クラスブロック
内側のブロック…メソッドブロック

メソッドブロック…コンピューターに対する指示、命令を記述する所。

クラス名…プログラムの名前を指定する単語。
public class の後に続いて書く。

javaのソースコードを記述したファイル(ソースファイル)を保存する時は
ファイルの名前は「クラス名.java」にする。

ステートメント…文。プログラムの実行時に上から順に１行ずつ処理されていく。

〜javaのコンパイルと実行の仕方〜

コンパイル… javac ソースファイル名

実行…java クラス名
